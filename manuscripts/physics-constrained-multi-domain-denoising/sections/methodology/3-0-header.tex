This section describes the experimental and computational framework used to train and evaluate PMDF-Net for physics-constrained multi-domain Lamb wave denoising. The network architecture integrates time-domain and time-frequency-domain branches with a physics-constrained loss that penalizes deviations from dispersive Lamb wave propagation physics. Training employs a paired laser ultrasonic dataset acquired from an intact aluminum plate, where input signals are recorded with limited shot averaging ($N = 16$) and target signals use high averaging ($M = 256$), yielding a nominal SNR improvement of approximately 12 dB. The training protocol uses stochastic gradient descent with early stopping on a held-out validation set. Generalization is assessed on separate aluminum specimens containing machined slots and edge notches. The following eight subsections cover: dataset description, signal preprocessing, network architecture, physics-constrained loss, training protocol, inference procedure, baseline methods, and evaluation metrics.
