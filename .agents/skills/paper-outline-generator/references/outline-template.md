# Paper Outline Templates

Ready-to-use templates for common paper types.

---

## Template 1: Standard Empirical Research

```markdown
## Paper Outline: [Topic]

**Research Type**: Empirical Study
**Target Journal**: [Journal Name]
**Word Target**: [X,000] words

### 1. Introduction
- [ ] **1.1 Background** - Establish the general research area and its importance
- [ ] **1.2 Problem Statement** - Identify the specific problem or gap
- [ ] **1.3 Literature Review** - Summarize key prior work and their limitations
- [ ] **1.4 Research Questions** - State RQ1, RQ2, etc.
- [ ] **1.5 Contributions** - List novel contributions of this paper
- [ ] **1.6 Paper Structure** - Overview of remaining sections

### 2. Methodology
- [ ] **2.1 Research Design** - Describe overall approach (quantitative/qualitative)
- [ ] **2.2 Participants/Sample** - Describe subjects, sample size, selection criteria
- [ ] **2.3 Data Collection** - Instruments, procedures, timeline
- [ ] **2.4 Data Analysis** - Statistical methods, software used
- [ ] **2.5 Validity/Rigor** - Measures to ensure internal/external validity

### 3. Results
- [ ] **3.1 Descriptive Statistics** - Overview of sample/dataset characteristics
- [ ] **3.2 Main Findings** - Results addressing each RQ with data support
- [ ] **3.3 Additional Findings** - Unexpected or secondary results
- [ ] **3.4 Statistical Details** - Effect sizes, confidence intervals, p-values

### 4. Discussion
- [ ] **4.1 Interpretation** - What the findings mean
- [ ] **4.2 Comparison with Literature** - How findings align with/differ from prior work
- [ ] **4.3 Implications** - Theoretical and practical significance
- [ ] **4.4 Limitations** - Constraints on validity and generalizability
- [ ] **4.5 Future Work** - Open questions and next steps

### 5. Conclusion
- [ ] **5.1 Summary** - Brief recap of research and key findings
- [ ] **5.2 Contributions** - Emphasize novel contributions
- [ ] **5.3 Closing** - Final thoughts on significance
```

## Template 2: Engineering / Technical Paper

```markdown
## Paper Outline: [Technical Topic]

**Research Type**: Technical / Engineering
**Target Journal**: [Journal Name]
**Word Target**: [X,000] words

### 1. Introduction
- [ ] **1.1 Background** - Technical context and motivation
- [ ] **1.2 Related Work** - Existing approaches and their limitations
- [ ] **1.3 Problem Definition** - Formal problem statement
- [ ] **1.4 Proposed Approach** - High-level description of your solution
- [ ] **1.5 Contributions** - Novel aspects and key innovations
- [ ] **1.6 Paper Structure** - Overview of remaining sections

### 2. Related Work
- [ ] **2.1 [Category 1]** - Prior work in [area], limitations
- [ ] **2.2 [Category 2]** - Prior work in [area], limitations
- [ ] **2.3 Comparison Table** - Side-by-side comparison of approaches

### 3. Proposed Method / System Design
- [ ] **3.1 System Architecture** - High-level design with components
- [ ] **3.2 Algorithm/Method 1** - Description and rationale
- [ ] **3.3 Algorithm/Method 2** - Description and rationale
- [ ] **3.4 Implementation Details** - Key design choices, tools, frameworks
- [ ] **3.5 Complexity Analysis** - Theoretical bounds if applicable

### 4. Experiments / Evaluation
- [ ] **4.1 Experimental Setup** - Datasets, hardware, baseline methods
- [ ] **4.2 Evaluation Metrics** - Metrics used and why
- [ ] **4.3 Main Results** - Performance comparison with baselines
- [ ] **4.4 Ablation Study** - Component-wise analysis
- [ ] **4.5 Sensitivity Analysis** - Parameter sensitivity, robustness

### 5. Discussion
- [ ] **5.1 Performance Analysis** - Interpretation of results
- [ ] **5.2 Comparison with Related Work** - Advantages and trade-offs
- [ ] **5.3 Limitations** - Scope constraints, assumptions
- [ ] **5.4 Future Work** - Potential improvements and extensions

### 6. Conclusion
- [ ] **6.1 Summary** - Brief recap of problem and solution
- [ ] **6.2 Contributions** - Key innovations
- [ ] **6.3 Impact** - Practical/theoretical significance
```

## Template 3: Literature Review / Survey

```markdown
## Paper Outline: [Topic] Literature Review

**Research Type**: Systematic Literature Review
**Target Journal**: [Journal Name]
**Word Target**: [X,000] words

### 1. Introduction
- [ ] **1.1 Background** - Importance of the research area
- [ ] **1.2 Need for Review** - Why a review is needed now
- [ ] **1.3 Review Objectives** - Research questions (RQ1, RQ2...)
- [ ] **1.4 Scope** - Boundaries and inclusion/exclusion criteria
- [ ] **1.5 Paper Structure** - Overview of remaining sections

### 2. Search Methodology
- [ ] **2.1 Search Strategy** - Databases, keywords, time period
- [ ] **2.2 Selection Process** - Inclusion/exclusion criteria
- [ ] **2.3 PRISMA Flow Diagram** - Study selection visualization
- [ ] **2.4 Quality Assessment** - How studies were evaluated
- [ ] **2.5 Data Extraction** - Information extracted from each study

### 3. Thematic Analysis / Synthesis
- [ ] **3.1 [Theme 1]** - [Description]
  - Study A: [Finding]
  - Study B: [Finding]
- [ ] **3.2 [Theme 2]** - [Description]
- [ ] **3.3 [Theme 3]** - [Description]

### 4. Findings
- [ ] **4.1 Overview of Literature** - Trends, quantity, quality
- [ ] **4.2 [Finding Category 1]** - Detailed synthesis
- [ ] **4.3 [Finding Category 2]** - Detailed synthesis
- [ ] **4.4 Research Gaps Identified** - What is missing

### 5. Discussion
- [ ] **5.1 Interpretation** - What the synthesis reveals
- [ ] **5.2 Comparison with Expectations** - How findings align with predictions
- [ ] **5.3 Implications** - For research and practice
- [ ] **5.4 Limitations** - Scope constraints, methodological notes
- [ ] **5.5 Future Research Agenda** - Recommended next steps

### 6. Conclusion
- [ ] **6.1 Summary** - Brief recap of review scope and key findings
- [ ] **6.2 Contributions** - Knowledge contributions of the review
- [ ] **6.3 Closing** - Final remarks
```

## Template 4: Short Conference Paper (4 pages)

```markdown
## Paper Outline: [Topic]

**Research Type**: Conference Paper
**Target**: [Conference Name]
**Length**: 4 pages

### 1. Introduction (0.5 page)
- [ ] **1.1 Motivation** - One paragraph on why this matters
- [ ] **1.2 Problem** - One paragraph on specific problem
- [ ] **1.3 Contribution** - Bulleted list of key contributions

### 2. Background / Preliminaries (0.5 page)
- [ ] **2.1 Key Concepts** - Minimum background needed
- [ ] **2.2 Related Work** - Brief prior work summary

### 3. Proposed Method (1.5 pages)
- [ ] **3.1 Overview** - High-level description
- [ ] **3.2 Technical Details** - Core approach with equations/algorithms
- [ ] **3.3 Key Innovations** - What's novel

### 4. Evaluation (1 page)
- [ ] **4.1 Setup** - Datasets, baselines, metrics
- [ ] **4.2 Results** - Key performance numbers
- [ ] **4.3 Analysis** - Why approach works/better

### 5. Conclusion (0.5 page)
- [ ] **5.1 Summary** - Brief recap
- [ ] **5.2 Future Work** - Optional next steps
```

## Customization Tips

### Adjusting Section Depth
- **Narrow topic** (single method improvement): Compress Related Work, expand Experiments
- **Broad topic** (new problem): Expand Introduction and Problem Definition
- **Application paper**: Add "Case Study" section, reduce theoretical depth

### Word Count Distribution
| Paper Type | Intro | RelWork | Method | Results | Discuss | Concl |
|------------|-------|---------|--------|---------|---------|-------|
| Empirical | 15% | - | 30% | 30% | 20% | 5% |
| Technical | 15% | 15% | 35% | 25% | 10% | 5% |
| Review | 10% | 15% | 20% | 30% | 20% | 5% |
